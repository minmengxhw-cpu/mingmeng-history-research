## 结语（约 150 字）

陈述档案集合的客观属性：2 篇 Hoover 档案为同一作者（张君劢）在 1947 年下半年致美国军政高层的私人信函，时间跨度 3 个月。其中 1947-11-01 信函时点位于民盟被宣布"非法"（1947-10-29）后约 7 天，为该事件相关的私人沟通档案。两篇档案均为 FRUS / CIA 公开档案所无的"民盟领导人致美方私人沟通"史料。
