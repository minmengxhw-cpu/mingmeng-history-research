# 证据卡片 006 · 1947-1949 海外民盟继续活动

> v2 七源互证版（2026-06-03 升级）· 补全 FRUS/DRNH/Hoover/Wilson 互证

## 核心问题

民盟在中国大陆遭禁（1947-10-27）后，南洋和香港相关分支并未立即从报刊视野消失。多源材料记录了**泛马来亚总部计划**、**海外分部公开声明继续运作**、**华侨政治组织与英殖民地监管之间的张力**——构成民盟史"1947-10 转入地下"叙事的关键反证。

## 研究价值

该证据链可以解释民盟"海外继续活动"的**三层组织空间**：
1. **香港作为总部 / 联络节点**（HathiTrust + Hoover + HKPRO）
2. **南洋作为公开舆论与侨社动员节点**（NewspaperSG + CIA）
3. **英殖民地政府提供监管视角**（HKPRO + NewspaperSG）

这是民盟海外史最复杂的事件链，可作为"民盟海外组织形态学"研究的核心样本。
