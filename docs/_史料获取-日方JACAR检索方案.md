## 一、JACAR 对本库的独特价值：补「民盟创建期 1941–1945」

JACAR 收录的是**日本战前·战中官方档案，时间下限约 1945 年（二战结束）**。这正好与本库现状形成互补：

- 现有 7 源（FRUS/DRNH/CIA/Hoover/HathiTrust/Wilson/NewspaperSG）**重心在 1946–1949**（政协、非法化、香港重建）；
- **1941–1945 民盟创建期**（中国民主政团同盟 1941 成立 → 1944 改名中国民主同盟）在现有源里相对薄。

> 民盟前身「中国民主政团同盟」1941 年 3 月成立于重庆，正值日本侵华战争中期。**日方军政情报系统对重庆国民政府治下的"第三势力""中间党派"必有观察记录**——这是其他六源都覆盖不到的、来自交战对方的同期视角。
> **结论：JACAR 的突破价值不在数量，而在补上"民盟最早五年"这段时间盲区的第三方（日方）视角。**

> 注意：日方战时文书多用「支那」指中国（贬称，史料原词，研究须保留原文、不改写）。

### 2.4 检索策略

1. **优先组合**：「民主政団同盟」单独检索（最专、误收最少）→ 再「第三勢力」+「重慶」→ 再人物名（張瀾/羅隆基）。
2. **旧字体优先**：日方档案用旧字体，「國/團/黨/鈞」等比新字体命中多。
3. **机构来源筛选**：JACAR 可按所藏机关筛——**外务省外交史料馆 + 防卫研究所（陆海军）** 两类对中国党派情报最集中。
4. **误收防线**：「民主同盟」宽词会命中战后日本本土及他国同名组织，须按"是否涉中国重庆方第三势力"人工判定（沿用 NewspaperSG 误收复核思路）。

## 三、抓取与入库流程

1. **前台检索确认命中**：用上表词在 JACAR 前台（www.jacar.archives.go.jp）逐词检索，记录返回的 **レファレンスコード（Ref. Code，档案唯一号）**、题名、年代、所藏机关、图像页数。
2. **建候选清单**：导出为 `data/jacar/candidates.csv`（字段：ref_code / 题名日文 / 年代 / 所藏机关 / 图像数 / 相关度初判）。
3. **下载图像**：JACAR 提供图像（JPG/PDF）公开下载，按 ref_code 批量取。
4. **OCR + 翻译**：日文档案 OCR（含竖排/旧字体，难度高于英文）→ 日文原文 → 中文翻译（标 `machine-draft`）→ 页码引用 → 来源卡。新 source platform 代码建议 `jacar`。
5. **接入七源**：入库后七源升级为含日方的更完整视角，**重点补 1941–1945 创建期的对位空档**。

> **技术提醒**：日文竖排 + 旧字体 OCR 质量是难点，建议先小批量（10–20 件高相关）验证 OCR+翻译流水线，再决定是否扩量。抓取脚本建议放 `scripts/probe/` 和 `scripts/ingest/`，复用现有 NewspaperSG 的图像→OCR→翻译模式。
