# 党政机关公文格式规范（项目内部执行参考）

> **依据**：中华人民共和国国家标准 **GB/T 9704-2012《党政机关公文格式》**
> （2012-06-29 发布，2012-07-01 实施，原 GB/T 9704-1999 同时废止）
>
> **适用范围**：本研究库今后所有对外发送的 `.docx` 报告，**默认按党政公文规范排版**；
> 学术研究包等纯学术体例的 `.docx` 可单独保留（标记 `_学术版`），
> 与公文版区分输出。

---

## 一、版面与纸张

| 项 | 规格 |
|----|------|
| 纸张 | **A4**（210 mm × 297 mm） |
| 页边距 | 上 **37 mm** / 下 **35 mm** / 左 **28 mm** / 右 **26 mm** |
| 单/双面 | 通常单面，长文双面（双面时奇数页右、偶数页左装订） |

> 默认 `Word/WPS` 中文版面设置即可对应。`python-docx` 用 `Cm(3.7)` / `Cm(3.5)` / `Cm(2.8)` / `Cm(2.6)`。

## 二、字体与字号

### 2.1 字号对应（磅值）

| 字号 | 磅值（pt）|
|------|----------|
| 二号 | **22** |
| 小二号 | 18 |
| 三号 | **16** |
| 小三号 | 15 |
| 四号 | 14 |
| 小四号 | 12 |
| 五号 | 10.5 |

### 2.2 各部分字体规范

| 用途 | 字体 | 字号 | 备注 |
|------|------|------|------|
| **公文标题** | 方正小标宋简体（华文中宋）| **二号** | 加粗，居中，分行不切割语义 |
| **发文字号** | 仿宋_GB2312 | 三号 | 如「沪盟字〔2026〕5 号」 |
| **主送机关** | 仿宋_GB2312 | 三号 | 顶格 |
| **正文** | **仿宋_GB2312** | **三号** | 行距 28 磅固定值 |
| **一级标题**（一、二、三、）| **黑体** | 三号 | |
| **二级标题**（（一）（二））| **楷体_GB2312** | 三号 | |
| **三级标题**（1.、2.、3.）| 仿宋_GB2312 | 三号 | **加粗** |
| **四级标题**（(1)(2)(3)、①②）| 仿宋_GB2312 | 三号 | 不加粗 |
| 附件标识 | 仿宋_GB2312 | 三号 | 「附件：」 |
| 发文机关署名 | 仿宋_GB2312 | 三号 | 右对齐 |
| 成文日期 | 仿宋_GB2312 | 三号 | 右对齐，用**阿拉伯数字**全标年月日（如 2026 年 5 月 16 日）|
| 抄送机关 | 仿宋_GB2312 | 四号 | |
| 印发机关 / 印发日期 | 仿宋_GB2312 | 四号 | |
| 页码 | 宋体或仿宋 | 四号 | 单面右下角；双面奇右偶左 |

### 2.3 字体 fallback（跨平台兼容）

`python-docx` 写入 `.docx` 时，East-Asian 字体设置应给出**多个候选**：

| 角色 | Windows | macOS | Linux（系统） |
|------|---------|-------|--------------|
| 仿宋 | FangSong_GB2312 / 仿宋 | STFangsong / 华文仿宋 | NSimSun / FangSong |
| 黑体 | SimHei / 黑体 | STHeiti / 华文黑体 / Heiti SC | Noto Sans CJK SC / WenQuanYi Zen Hei |
| 楷体 | KaiTi_GB2312 / 楷体 | STKaiti / 华文楷体 / Kaiti SC | AR PL UKai CN / cwTeX Q Kai |
| 小标宋 | 方正小标宋简体 / 方正小标宋_GBK / 华文中宋 | STZhongsong / 华文中宋 | Noto Serif CJK SC Bold |

实战中 `python-docx` 用以下两层设置：
```python
run.font.name = "FangSong"                                  # ascii fallback
run._element.rPr.rFonts.set(qn('w:eastAsia'), '仿宋')      # East-Asian
```
打开方所在系统会自动 fallback 到本机第一个匹配的字体。

## 三、段落与行距

| 项 | 规格 |
|----|------|
| 正文行距 | **固定值 28 磅** |
| 段前段后 | **0 行**（默认） |
| 首行缩进 | **2 字符**（约 7.4 mm，即 0.74 cm 或 22 pt）|
| 对齐 | 正文**两端对齐**，标题居中或顶格 |
| 标点 | 中文标点（句号「。」逗号「，」） |

## 四、结构要素（自上而下）

```
┌─────────────────────────────────────────┐
│           [发文机关] 红色版头             │ ← 红色机关名（仅有正式行文使用）
│           [发文字号] 三号仿宋             │
│  ─────────────────────────  ← 红色分隔线 │
│                                          │
│          《文件标题》(二号小标宋)           │ ← 居中，可分多行
│                                          │
│  主送机关：(三号仿宋顶格)                 │
│                                          │
│      正文第一段……(三号仿宋首行缩进 2 字)   │
│                                          │
│      一、一级标题（黑体三号）              │
│      （一）二级标题（楷体三号）            │
│      1. 三级标题（仿宋三号加粗）           │
│                                          │
│  附件：1.附件标题                         │
│        2.附件标题                         │
│                                          │
│                       发文机关署名 (三号) │
│                       2026 年 5 月 16 日  │ ← 成文日期居右
│                       [印章]              │
│                                          │
│  ─────────────────────────              │
│  抄送：xxx、xxx、xxx 等。(四号仿宋)        │
│  ─────────────────────────              │
│  xxx 印发                  2026 年 5 月    │
└─────────────────────────────────────────┘
```

> **非正式行文**（如**调研报告 / 研究专题 / 工作汇报**）通常**省略红色版头与印章**，
> 只保留标题、正文、署名、日期、抄送等结构。
> 本研究库导出的报告默认采用**非正式行文体例**——无红头无印章，
> 但字体字号行距严格按公文规范。

## 五、常见文种（小班机关工作语境）

| 文种 | 用途 |
|------|------|
| 通知 | 上对下，知会事项 |
| 报告 | 下对上，汇报情况 |
| 请示 | 下对上，请求指示批准 |
| 函 | 平行机关协商工作 |
| 决定 | 重大事项决策 |
| 决议 | 经会议通过的决策 |
| 纪要 | 会议讨论记录 |
| **简报 / 盟讯** | 内部刊物，工作动态汇编 |
| **调研报告** | 专题调研成果 |
| **讲话稿 / 发言稿** | 领导讲话底稿 |

研究库**主输出文种**：
- **调研报告**（学术专题转化为机关汇报）
- **简报 / 盟讯**（短篇 1-4 页工作动态）

## 六、数字使用规范

| 类型 | 写法 |
|------|------|
| 公历年月日 | 阿拉伯数字：2026 年 5 月 16 日 |
| 数量 | 阿拉伯数字：3 篇文档 / 21 段史料 |
| 序号 | 阿拉伯数字：第 1 章 |
| 古汉语 / 习惯用语 | 中文：一日、二十四史、五四运动 |
| 大数字 | 加千位分隔符：14,109 篇 |
| 范围 | 起讫年用「—」连接：1941—1949 年（中文长破折号） |

## 七、引文与参考文献

| 项 | 规则 |
|----|------|
| 直接引文 | 中文双引号「」嵌套时用『』 |
| 长引文 | 单独成段，左右缩进 4 字符 |
| 引文出处 | 脚注或文末参考文献 |
| 参考文献 | 按 **GB/T 7714-2015**（与学术体例兼容）|

学术档案引用本研究库时建议标注：
> 民盟历史文献研究库 / FRUS Vol.X Document Y / CIA-RDP82-... (archive.org 镜像) / 2026
