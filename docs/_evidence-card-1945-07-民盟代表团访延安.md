# 1945 年 6—7 月赫尔利接见 → 民盟代表团访延安事件·跨源证据卡片集

> 民盟历史文献研究库 · 跨源证据卡片库 · 第 003 号事件
> 撰文日期：2026-06-01 · 撰文：研究平台项目组

## 0. 本卡片集的方法学依据 + 事件定位

按方法论第 2 条「孤证不举，互证优先」+ 第 5 条「证据卡片」格式落实。

**本事件的核心方法学价值**：1945 年 7 月 1 日**美军机载民盟领导人黄炎培、左舜生、
褚辅成等赴延安**与毛泽东会谈，是抗战胜利前 6 周民盟与中共关系的标志性事件。
DRNH 国府打压方档案对此事的**密集监视记录**（戴笠呈件 4 件 + 钱大钧呈件 + 军统
侦察电台 + 蒋本人电令 + 叶镇宇函蒋），是研究"民盟—中共—美方—国府"四方关系的
关键档案集合。

**跨源对位特点**：
- DRNH 国府监视档案密集（7 件）
- FRUS 同期赫尔利大使报告系列（论文 v3 §3.4 已述，d286 + d290 + d295）
- 因为是 1945 年 6—7 月（CIA 1947 才有，HathiTrust 1946 才有，Wilson 1945-10 才开始），
  本事件**仅 DRNH + FRUS 双源对位**

**卡片字段**（沿用第 001、002 号事件标准）。

## 五、卡片集总览与方法论小结

### 5.1 本事件的档案分布

| 源 | 时段内档案数 | 主要时点 |
|---|---:|---|
| DRNH | 7 篇 | 1945-06-23 / 06-30 ×2 / 07-01 ×2 / 07-05 / 07-07 |
| FRUS | 3 篇（论文 v3 §3.4） | 1945-06-23 d286 + 06-27 d290 + 07-10 d295 |
| HathiTrust / CIA / Wilson / Hoover | 0 | 时段过早 |
| **合计** | **10 篇** | DRNH 高密度 + FRUS 对位 |

### 5.2 本卡片集的核心发现

1. **同日双源呈报**（T02 + T03）：1945-06-30 戴笠（军统）+ 钱大钧（侍从室）两线情报系统同日呈报"赫尔利接见民盟"事件——**事件重要性极高的标志**
2. **代表团全名单的两档对照**（T04 + T06）：T04 戴笠当日呈报仅列 3 人（黄左褚），T06 军统侦察电台 4 天后补全 6 人（加冷遹+傅斯年+章伯钧）——研究**国府情报时效与准确度**的微观档案
3. **三方信息同日并陈在蒋档**（T05）：宋子文与苏联谈军火 + 赫尔利接见民盟 + 美军机赴延安——蒋同日意识到"民盟+美方+中共+苏联"四方关系交错
4. **国府对民盟内部会议的监视**（T07）：1945-07-07 民盟在重庆听取访延报告——国府情报系统通过叶镇宇渠道获知内部会议内容
5. **DRNH + FRUS 双源对位完整**：T01 + T04 + T05 三档与 FRUS 三档（d286 / d290 / d295）形成"美方授权（送飞机）→ 国府监视（戴笠呈件）→ 美方报告（赫尔利系列）"完整档案链
6. **傅斯年作为非民盟学者同机赴延安**（T05 + T06）：是该事件的重要"中性观察者"——傅斯年后著《延安观感》是关键二手参考

### 5.3 按方法论的研究产出

本卡片集可直接支持以下研究问题：

- **民盟早期对中共的态度问题**：1945-07 民盟代表团访延安、听取报告——民盟内部对中共的实质判断？
- **美方对民盟的政治期望问题**（T02 + T03）：赫尔利期望民盟"转圜国共僵局"，美方支持力度的边界？
- **国府情报系统的层次问题**：戴笠（军统）+ 钱大钧（侍从室）+ 军统侦察电台 + 叶镇宇 4 条情报线对同一事件的不同视角
- **代表团人员构成问题**：傅斯年（非民盟学者）+ 民盟 5 人为何同机？
