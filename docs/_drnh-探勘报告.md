## 一、API 调通技术备忘

- **核心发现**：搜索 URL 走 `/index.php?act=Archive/search/<base64m_json>[/<start>-<end>]`
- **Base64M**：标准 Base64 但把 `/` 替换为 `*`（系统自定义）
- **JSON payload**：`{"accnum":"0","query":[{"field":"_all","value":"<關鍵詞>"}]}`
- **翻页**：URL 末尾追加 `/<start>-<end>`（每页支持 20/50/100，本次用 100）
- **元数据 100% 访客开放**（无需会员）
- **图像可下载** 但有「請登入」绿色水印（访客身份）
- **绕过 captcha**：首次 GET `index.php?act=Archive` 拿 cookie 后直接构造 search URL，**无需提交 Turing 验证码**

## 四、A 档分布

### 按全宗（292 条）
- 蔣中正總統文物：占比 ~70%（含戴笠呈件、保密局密报、蔣的批示等最高层档案）
- 國民政府：占比 ~16%（含 1946 山东四川青海三省参议会反驳民盟系列）
- 陳誠副總統文物：占比 ~9%（含国大情报系列）
- 外交部：占比 ~5%（含驻外使馆对民盟在港活动的报告）
- 其他全宗（軍委侍從室、閻錫山史料等）少量

### 按年份
- **1946 年峰值** ~110 条（政协会议 + 国共谈判破裂 + 民盟与国民党摩擦升级）
- **1947 年** ~70 条（民盟「非法」事件 + 杜斌丞案 + 学潮中民盟视角）
- 1945 年 ~50 条（抗战胜利前后 + 民盟在重庆活动）
- 1948 年 ~30 条（民盟在港复盘 + 改组）

### 按提供方式
- 「數位檔／線上閱覽」: ~180 条 — 访客可下水印图，会员可看原图
- 「數位檔／臺北閱覽室」: ~95 条 — 需赴台北
- 其他: ~17 条

## 八、本次教训复盘（再补强）

近史所探勘犯过两次「汇总数代结论」错误（35 条线上误命中 + 胡适档案 21 条全核心），这次国史馆探勘**严格按「题名直接命中 + 时段对齐 + 全宗判断」四对齐审视**，A 档抽样 12 条全部命中民盟史核心。但 B 档仍有 183 条未深入个体审视（保留为「需现场或深入考察」状态），未来注入本平台时应继续逐条精化。

**核心方法**：永远不能只看 count，必须看每条的题名（不是关键词，是完整题名）。
