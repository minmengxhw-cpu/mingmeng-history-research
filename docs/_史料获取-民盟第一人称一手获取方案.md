## 六、收录与口径纪律

1. **一手判定**：本人日记/文集/书信、本组织机关报刊原文、本组织自编文件汇编 = 一手，入正式层；他人传记、回忆文章、政协文史资料（回忆性）、官网介绍 = 二手，入参考层，**不作结论依据**。
2. 中文一手**不翻译**，仅繁→简、OCR 校；保留原文与出处（报名/期号/日期/页）。
3. 与境外七源**对位**：《光明报》建盟宣言 ↔ 各国档案对民盟成立的观察；民盟方声明 ↔ 国府/美方记录——首次让"民盟自己说"与"五方凝视"同台。
4. 政治表述存疑标"待核"；涉章程性文件按民盟正式表述，不自创口径。
