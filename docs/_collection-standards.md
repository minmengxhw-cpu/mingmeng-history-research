# 本库收录标准与排除标准

> 民盟历史文献研究库 · 编辑准则与方法论说明
> 最近修订：2026-05-28

本页说明本研究库的档案收录原则、各源收录边界、排除标准、前台展示分级体系，以及可追溯性原则。这是面向研究者的"编辑准则"页面——研究者在引用本库档案前，应先了解每篇档案为何被收录、被分到哪一档、哪些材料已被剔除及其理由。

---

## 一、本库的研究定位

本库是**中国民主同盟（民盟）史的境外一手档案研究库**，聚焦 1941—1950 年民盟史的六个境外一手档案源：

| 源 | 视角 | 收录规模 |
|---|---|---|
| FRUS | 美方公开外交 | 273 直接命中 + 26 间接命中 |
| DRNH | 国民政府打压方内部 | 抓取 1138 → 展示 364 → 精读 282 |
| CIA | 美方解密情报 | 入库 102 → 前台 76 |
| HathiTrust / IA | 港埠公开舆论 | 54 期港媒（38 期含民盟实质内容）|
| Wilson Center | 苏方与东欧 | 24 篇 |
| Hoover | 民盟创始人致美方私函 | 2 件（张君劢致 Wedemeyer / Marshall）|

**收录原则的两条铁律**：

1. **一手档案优先**：本库只收录境外一手档案（原始外交电报、情报报告、政府内部呈件、当时公开报刊、私人信函、苏方解密档案），**不收录二手研究、回忆录、转述、综述类材料**。
2. **民盟实质相关**：档案必须与中国民主同盟（China Democratic League / Chinese Democratic League / 中国民主同盟 / 中国民主政团同盟）有实质组织关系或对民盟史有直接史料价值，**仅名称相似而无实质关系的材料一律剔除前台**。

## 二、收录范围（保留标准）

### 2.1 明确保留

满足以下任一条件的档案，明确保留：

- 明确指向 **China Democratic League / Chinese Democratic League / 中国民主同盟 / 中国民主政团同盟 / 民盟中央**
- 涉及**民盟核心人物**：张澜、罗隆基、章伯钧、张君劢、左舜生、黄炎培、沈钧儒、张申府、史良、李公朴、潘光旦、张东荪、梁漱溟、杜斌丞等
- 记录**民盟在香港 / 上海 / 北平 / 昆明 / 重庆等地的组织活动**
- 国民政府 / 美方 / 苏方对民盟的**监视、情报、批阅、决策档案**
- 民盟领导人**致外方的公开声明、备忘录、私函**

### 2.2 谨慎保留（标注海外/南方分支）

下列档案虽含"Democratic League"在异地，但已审定为中国民盟海外/南方系统，**保留前台 + 标注分支属性**：

- **South China Democratic League**（华南民盟，民盟南方系统）——如 CIA `r000300170001-3`（李章达 + 丘哲签署，命令来自上海民盟总部）
- **China Democratic League Selangor Branch**（雪兰莪分支，民盟马来亚海外华侨分支）——如 CIA `r000100470002-1`、`r000200030007-3`
- 翁士良（华南民盟主席）赴港行程等华南民盟领导人活动档案

判定原则：**命令来源、人员关系、组织隶属可追溯到中国民盟总部（上海/香港）的，保留；否则剔除**。

## 三、排除标准（剔除前台）

### 3.1 名称相似但与中国民盟无组织关系（一律剔除前台）

民盟史档案检索的最大噪声来源是"Democratic League"在不同语境下指向不同组织。下列**名称相似但与中国民主同盟分立**的组织，一律剔除前台：

| 类型 | 组织名 | 说明 |
|---|---|---|
| 朝鲜 | 朝鲜民主同盟 / Korean Democratic League | 朝鲜半岛政治组织，与中国民盟无关 |
| 台湾 | Taiwan Democratic League / New Democratic League on Taiwan（台湾新民盟）| 台籍政治组织，与中国民盟分立 |
| 台湾 | Formosan League for Re-Emancipation（台湾再解放联盟）| 台湾独立运动组织 |
| 台湾 | Democratic League for Taiwan Autonomy / 台湾民主自治同盟 | 与中国民盟分立的台籍组织 |
| 东南亚 | 缅甸 / 越南 / 印尼 / 马来亚 / 泰国共产党与华人组织 | 仅名称相似，无民盟组织关系 |
| 日本 | 1948 日本共产势力相关档案 | 名称巧合 |

### 3.2 远离民盟史核心时段

1956 年后远离民盟史核心期（1941—1950）的材料，如 CIA 中央情报简报系列、1957 亚非团结会议、1982 FACTBOOK 等，剔除前台（保留内部检索）。

### 3.3 二手 / 转述 / 综述类

不属于一手档案的材料（二手研究、回忆录、综述、转述报道）不予收录。

### 4.2 DRNH：抓取 1138 行 → 剔除 101 条 → 展示 364 篇 → 精读 282 篇

DRNH 抓取的 1138 行原始记录中，剔除 101 条无关档案（含同名组织、误匹配、与民盟无实际关联的档案），保留 364 篇有效档案。其中 282 篇进入三层校订分层与 LLM 精读：

| 校订分层 | 篇数 | 说明 |
|---|---:|---|
| 重点校订（A 档）| 136 | 直接命中民盟 + 核心人物 + 核心事件，研究核心引用范围 |
| 常规校订 | 13 | 中等优先级 |
| 背景保留（B 档）| 133 | 民盟相关但非核心的背景材料 |

282 篇 LLM 精读的民盟相关度自评：核心 124 / 相关 39 / 背景 110 / 误命中 9。

### 4.3 HathiTrust：54 期港媒（38 期含民盟实质内容 + 16 期空）

54 期港媒整期 OCR 经关键词预筛后，38 期含至少 1 个民盟相关段落，16 期 OCR 无关键词命中（多为 OCR 质量差期次或 1950 年报道兴趣减少的期次）。空期次保留入库但前台标注"本期无民盟实质报道"。

### 4.4 FRUS / Wilson / Hoover

- **FRUS**：273 篇直接命中（标题/正文明确含民盟）+ 26 篇间接命中（民盟为旁证提及）
- **Wilson**：24 篇全部保留（苏方档案稀缺，即使"仅含背景"也保留作语境）
- **Hoover**：2 件全部保留（民盟创始人主动陈述档案极珍贵）

## 五、前台展示分级体系

本库对所有档案按"民盟相关度"分级，前台用标签呈现。研究者引用时应遵循分级优先级：

| 分级标签 | 含义 | 研究使用建议 |
|---|---|---|
| **核心文献** | 直接命中民盟 + 核心人物 + 核心事件 | 研究核心引用范围 |
| **相关文献** | 民盟为重要内容之一，但非全篇主体 | 辅助引用 |
| **人物关联** | 通过民盟人物间接关联，非民盟组织活动 | 人物研究参考 |
| **背景材料** | 民盟相关但非核心的背景档案 | 背景查证，不作主要引用 |
| **前台不展示** | 名称相似但与中国民盟无关 / 远离时段 / 误收 | 已从前台移除，仅内部检索可见 |
| **误收已剔除** | 经复审确认误命中，已标注剔除理由 | 不应使用 |

DRNH 平台另有专门的三层校订分层（重点校订 / 常规校订 / 背景保留），对应 A / B / C 档。

## 六、可追溯性原则

本库剔除档案时**不删除原始数据**，只在分类层把 `grade` 改为"前台不展示"：

1. **数据保留**：剔除的档案 OCR 文本、影像、元数据全部保留在数据库与文件系统
2. **前台过滤**：所有前台路由（搜索、栏目页、列表、年表、人物索引等）自动过滤"前台不展示"档案
3. **理由记录**：每篇剔除档案标注剔除组别（A/B/C/D/E）与具体理由
4. **可重审**：内部检索、学术回溯、后续重审仍可访问被剔除档案，剔除决定可被复核与推翻

这一原则确保：**收录边界的每一次调整都是可追溯、可复核、可推翻的**，而非不可逆的数据删除。

被剔出前台的档案清单（按平台分组、含逐篇剔除理由）见 [已剔除清单页](/excluded)。

## 七、LLM 辅助精读的认识论声明

本库各平台的学术论文（FRUS / CIA / DRNH / HathiTrust / Wilson / Hoover）均基于 LLM 辅助逐篇精读（DeepSeek v4-flash，结构化 JSON 输出，低 temperature）。研究者使用时需注意：

1. LLM 精读的民盟相关度评级、事件分类、人物识别**均为机器归纳**，非档案馆官方分类
2. 核心事实陈述是 LLM 对档案原文/元数据的归纳，可能丢失原档微妙措辞
3. **所有正式学术引用应以档案原文为准**（FRUS 原始电报 / CIA OCR 英文 / DRNH 内容描述繁体原文 / 港媒 OCR / Wilson 英文译稿 / Hoover 信函原文），本库论文与精读摘要仅作研究定位辅助
