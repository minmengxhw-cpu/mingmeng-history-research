# 六源对照档案体系：1941—1950 年中国民主同盟史的境外一手档案研究框架

> 民盟历史文献研究库 · 平台学术论文系列 · 总论
> 撰文日期：2026-05-26 · 撰文：研究平台项目组（机器初拟稿，待人工校订）

## 摘要

本文以民盟历史文献研究库收录的六大境外一手档案源（FRUS / CIA / Wilson Center / Hoover Institution / HathiTrust / 台北 DRNH）合计 821 篇档案 / 1059 个页段为研究对象，构建研究 1941—1950 年中国民主同盟史的"六源对照档案体系"。文章在六篇平台分论文（FRUS / CIA / DRNH / HathiTrust / Wilson / Hoover）基础上，做跨平台综合论述，呈现六源在视角、内容、时段、密度四个维度上的互补关系，提出"六源对照"研究方法学及其在民盟"非法化"事件（1947 年 10—11 月）研究中的完整应用样本。

**关键词**：中国民主同盟 · 民盟史 · 境外档案 · 多源对照 · 档案学 · 史料学

## 一、研究问题与档案体系的构建

中国民主同盟（民盟，China Democratic League）作为中国八大民主党派之一，从 1941 年 3 月在重庆成立至 1949 年 10 月新政权建立期间，其活动在多国档案系统中留下了丰富的记录。但这些档案分散于不同来源，研究者难以做系统对照研究。

本研究平台收录的"中国大陆境外一手档案"分布于六个独立来源：

| 平台 | 来源类型 | 文档数 | 页段数 | 视角 |
|---|---|---:|---:|---|
| FRUS | 美方公开外交档案集 | 299 | 416 | 美方公开外交 |
| 台北 DRNH | 国民政府档案 | 364 | 364 | 国民政府打压方内部 |
| CIA | 美方解密情报 | 78 | 102 | 美方情报系统 |
| HathiTrust | 港埠英文报刊 | 54 | 54 | 港埠公开舆论 |
| Wilson Center | 苏联及东欧档案 | 24 | 121 | 苏方与东欧 |
| Hoover | 民盟创始人致美方私函 | 2 | 2 | 民盟方→美方 |
| **合计** | | **821** | **1059** | |

> 注：CIA 原始抓取 102 篇，剔除 24 篇与中国民盟无关的同名组织档案后保留 78 篇；DRNH 原始抓取 1138 行，二次复核剔除 101 条后保留 364 篇。

本文研究问题：（1）六源在民盟史研究中的互补关系如何？（2）"六源对照"方法学如何在具体研究场景中应用？（3）平台对六源的工程化处理（OCR / 翻译 / 索引 / 跨源印证）为研究者提供了哪些可重复使用的研究工具？

## 二、六源在民盟史研究中的互补关系

### 2.1 视角维度的互补

六源覆盖民盟史研究的六个独立观察视角：

```
                      民盟（研究对象）
                     /              \
              对外发声               内部决策
              /  |  \              /  |  \
        Hoover  港媒HT  FRUS    DRNH  CIA  Wilson
       民盟方→  公开    美方公开  国民政府  美方     苏方
       美方    舆论场    外交    内部     情报     视角
       私函           （基线源）  决策    系统
```

- **公开档案**：FRUS（美方公开）、HathiTrust（港埠公开舆论）
- **半公开档案**：CIA（美方解密情报）、DRNH（国民政府档案，台湾国史馆开放）
- **学术档案**：Wilson（CWIHP 学者整理的苏方档案）、Hoover（学术机构现场调档）
- **民盟方主动输出**：Hoover（民盟创始人致美方私函）
- **民盟外部观察**：其余五源

### 2.2 时段维度的覆盖

| 平台 | 主要时段 | 时段特点 |
|---|---|---|
| FRUS | 1943—1950 | 8 年完整覆盖，按年度卷出版 |
| DRNH | 1945—1949 | 5 年密集覆盖，含按事件呈报 |
| CIA | 1946—1953 | 7 年覆盖，1946—1951 为高峰 |
| HathiTrust | 1946—1950 | 5 年近均匀时序档案 |
| Wilson | 1945 + 1948-12 至 1949-12 + 1958 | 集中于 1949 年高层通信 |
| Hoover | 1947-07 + 1947-11 | 2 个时点 |

FRUS 与 HathiTrust 提供完整时间序列；DRNH 与 CIA 提供事件密度档案；Wilson 与 Hoover 提供关键时点专项档案。

## 三、"六源对照"方法学：以 1947 年民盟"非法化"事件为研究样本

### 3.1 事件背景与档案场

1947 年 10 月 28 日国民政府内政部发言人正式宣布中国民主同盟为"非法团体"，要求该团体停止活动；同月底至 11 月初，国民党警察接管民盟总部，民盟主席张澜被警察"保护性看管"于上海寓所；11 月 5 日左右，民盟自行宣布解散。

该事件是民盟史研究中**最被密集观察的事件**之一，本平台六源对此事件的档案分布如下：

| 档案源 | 篇数 | 档案性质 |
|---|---:|---|
| DRNH 保密局呈件 | ~5-7 | 国民政府打压方内部决策档案 |
| HathiTrust China Mail | 5 期 | 港埠公开舆论场连续报道 |
| FRUS 1947v07 卷 | ~10 篇 | 美方驻华使馆电报 |
| CIA 解密情报 | ~3-5 篇 | 美方情报系统观察 |
| Hoover 张君劢致 Marshall | 1 篇（1947-11-01）| 民盟方→美方私人申诉 |
| **合计** | **~25 篇** | **五源完整覆盖** |

### 3.2 五源完整对照档案场

研究者可通过本平台五源对照档案，重建该事件的完整研究档案场：

**3.2.1 国民政府内部决策链（DRNH）**

DRNH 含一系列保密局呈件，记录国民政府从"内部讨论取缔"到"宣布非法"到"民盟解散"的完整内部决策链。其中 1947-11-05 保密局呈"民盟留沪人员姓名及疏散情形"含民盟成员撤离名单详细记录；1947-11-12 薛岳汇呈"民盟解散后首要分子动态"情报七则；1947-11-19 薛岳汇呈"民盟西南总支部否认解散"情报三则。

**3.2.2 港埠公开舆论场（HathiTrust）**

China Mail 在 30 天内连续 5 期报道：
- [1947-10-06 NPCM19471006](/doc/hathi-ia:NPCM19471006)：早期信号"上海方面已有取缔民盟内部讨论"
- [1947-10-24 NPCM19471024](/doc/hathi-ia:NPCM19471024)：民盟解散前夜相关信息
- [1947-10-29 NPCM19471029](/doc/hathi-ia:NPCM19471029)：译载国民政府内政部声明全文
- [1947-10-30 NPCM19471030](/doc/hathi-ia:NPCM19471030)：详载警察接管民盟总部、张澜被监视过程
- [1947-11-05 NPCM19471105](/doc/hathi-ia:NPCM19471105)：张澜将在 48 小时内宣布民盟自行解散

**3.2.3 美方驻华使馆观察（FRUS）**

FRUS 1947v07 卷美方对民盟"非法"事件的连续电报（约 10 篇）。研究者可通过 `volume_id=frus1947v07` + `date 1947-10` 至 `1947-11` 筛选定位。

**3.2.4 美方情报系统记录（CIA）**

CIA 解密档案对该事件的情报报告（约 3-5 篇）。

**3.2.5 民盟方对美方的直接申诉（Hoover）**

[hoover:CC-1947-11-01-marshall](/doc/hoover:CC-1947-11-01-marshall)：民盟创始人张君劢在民盟"非法"后约 4 天致美国国务卿 Marshall 的私人信函，含张君劢对该事件的个人陈述与对美方政策的呼吁。

### 3.3 五源对照的研究价值

通过五源对照，研究者可在民盟"非法"事件研究上获得：

- **完整决策链还原**：国民政府从内部讨论 → 宣布非法 → 接管总部 → 民盟解散 → 善后处理的连续档案
- **多视角现场记录**：美方观察、港埠舆论、民盟自身申诉的同时档案
- **时间窗口精细化**：30 天内 25 篇档案，平均每 1.2 天 1 篇
- **可学术引用性**：FRUS / HathiTrust / Hoover 均可经永久 URL 引用；DRNH / CIA 可经平台 doc_key 与原档典藏号溯源

这是民盟史研究中**少见的多源完整对照档案场**——其他多数事件最多 2-3 源对应。

## 四、平台工程化对研究的支持

本研究平台对六源档案做了以下工程化处理，为研究者提供可重复使用的研究工具：

### 4.1 全文索引（FTS5）

平台对全部 1059 个页段建立 SQLite FTS5 全文索引，支持英文 + 中文 + 繁简自动展开 + trigram 中文分词。研究者可通过 [/search](/search) 路径做关键词跨源检索，结果按平台分类、按时间排序。

### 4.2 机器翻译

平台对全部 1059 个页段做了机器中文翻译，状态分布：
- `human-reviewed`：人工已审，可直接学术引用
- `machine-draft-review-needed` / `machine-draft-local-review-needed`：机器初稿，需复审
- `pending-deepseek-retranslate`：排队等高质量重译

正式学术引用应回溯原文。机器翻译主要服务于全文检索与快速阅读。

### 4.3 三种学术引用格式生成

平台为每篇档案生成三种学术引用格式（GB/T 7714 / Chicago / BibTeX），通过 [/cite/<page_id>](/cite/) 路径获取。

### 4.4 跨源印证查询

详情页支持基于"本件日期 + 相关人员 + 关键事件"的跨平台档案对照查询（时间窗口过滤可调）。研究者在阅读单篇档案时可一键定位其他源同期档案。

### 4.5 校订分层与质量分诊

平台已建立多个工作台：
- DRNH 校订分层队列（`drnh_review_layers.csv`，282 条）
- 翻译质量分诊（`translation_quality_issues.csv`）
- HathiTrust 质量分诊（`hathitrust_quality_triage.csv`）
- CIA 译文修复工作台（`refine_cia_translation_residue.py`）

## 五、研究使用方法学总结

基于六平台分论文与本总论的论述，本平台对民盟史研究者的方法学建议如下：

**5.1 多源对照优先**。研究民盟史关键事件时，**不应仅依赖单一档案源**。建议研究者通过本平台跨源印证查询功能，定位事件相关的全部档案源，做多视角对照分析。

**5.2 按基线源—密度源—独有源分层使用**：
- **基线源**：FRUS（公开学术档案，最易引用）
- **密度源**：DRNH（国民政府事件呈报）、HathiTrust（港媒连续报道）
- **独有源**：CIA（民盟海外网络情报）、Wilson（苏方独立观察）、Hoover（民盟方对美方私函）

**5.3 翻译复审与原文回溯**。本平台机器翻译可用于全文检索与快速阅读，但**正式学术引用应回溯原文**：FRUS / Wilson 回溯英文译稿；HathiTrust / CIA 回溯英文 OCR；DRNH 回溯案由扫描件；Hoover 回溯英文信函原件。

**5.4 关注质量分诊与校订状态**。研究者使用单篇档案时建议先查看其在平台的校订状态与质量分诊标注，避免引用低质量译稿。

**5.5 利用平台工程化工具**。全文索引、跨源印证查询、引用格式生成、校订分层队列等工具均可提升研究效率，避免研究者重复做档案整理工作。

## 六、结论

本研究平台收录的六源 821 篇中国大陆境外一手档案，构成研究 1941—1950 年中国民主同盟史的"六源对照档案体系"。六源在视角、内容、时段、密度四个维度上构成互补关系：FRUS 为公开学术基线源；DRNH 为国民政府打压方内部决策档案；CIA 为美方情报系统记录；HathiTrust 为港埠公开舆论场连续报道；Wilson 为苏方与东欧独立观察；Hoover 为民盟创始人致美方私人沟通。

特别在 1947 年民盟"非法化"事件研究中，五源（DRNH + HathiTrust + FRUS + CIA + Hoover）形成约 25 篇 30 天密集对照档案场，是民盟史研究中可全维度还原的少数时段之一。

本平台对六源做了全文索引、机器翻译、引用格式生成、跨源印证查询、校订分层等多重工程化处理，为研究者提供了可重复使用的研究工具。后续工作将围绕：（1）扩展外部档案获取（含 Kew 英国国家档案馆 FO 371、HKPRO 香港政府档案、中研院近代史研究所档案、Hoover 扩展调档等）；（2）提升机器翻译质量；（3）完善跨源印证查询的精度与召回率三方面继续推进。
