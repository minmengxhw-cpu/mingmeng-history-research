## 引言：DRNH 364 篇档案的基本面（约 400 字）

陈述档案的客观属性：
- 档案来源：台北国史馆「檔案史料文物查詢系統」（ahonline.drnh.gov.tw）
- 入库数量：364 篇（1138 行原始抓取 → 严格二次复核剔除 101 条无关档案 → 364 篇有效）
- 时间分布：主要集中在 1945-1949 年，少量延伸至 1950s
- 档案类型（按 OCR 抽样分布）：
  - **蒋档（蒋中正总统文物）**：含蒋介石批阅件、亲笔签注、内部电报
  - **戴笠呈件**：情报系统对民盟动态的直接呈报
  - **保密局呈报**：1946-1947 国统区各地民盟活动情报
  - **薛岳、邱清泉、郑介民等战区呈报**：地方军政对民盟的观察
  - **张嘉璈、宋子文等政要档案**：与民盟领导人接触的内部记录
- 入库形态：访客水印图（带 DRNH 水印的扫描件）+ 案由元数据 + 「机器初拟」案由学术导读（265 份已生成）
- 校订分层（已建 `drnh_review_layers.csv`）：
  - 重点校订：136 篇（直接命中民盟 + 民盟核心人物 + 核心事件）
  - 常规校订：13 篇
  - 背景保留：133 篇
  - 合计 282 篇进入分层（剩 82 篇为高频通用档）

## 结语（约 300 字）

陈述档案集合的客观属性：DRNH 364 篇档案为国民政府最高决策层（蒋介石本人 + 戴笠 + 保密局 + 各战区长官）对民盟的内部档案集合。该集合在档案类型上与 FRUS（美方外交电报）、CIA（美方情报系统）、Wilson（苏方档案）形成完全不同的视角——为**国民政府内部决策视角**。其中**保密局呈蒋档（约 25 篇，第三章）**是 DRNH 中民盟相关性最高的子集，扩写时建议作为综述重心。

整体上，DRNH 与 HathiTrust（港媒）、FRUS（美方）、CIA（美方情报）、Wilson（苏方）、Hoover（民盟领导人致美方私函）共同构成"民盟史多源对照"档案体系；DRNH 是其中**唯一来自打压方（国民政府）内部视角**的档案源。
