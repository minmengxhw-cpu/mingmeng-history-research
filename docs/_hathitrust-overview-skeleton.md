> 第二版骨架，按小班反馈调整：放弃"先有历史叙事再套档案"的写法，改成**从档案事实出发**的学术叙述。
> 章节标题描述档案内容（非历史评价），每篇档案的提及来自其实际记录的事实，不掺"标志着 / 开启了 / 体现了"等预设结论词。

---

## 写作原则（v2 调整）

1. **档案先于叙事**：每个论断必须能在档案原文中找到对应文字依据
2. **客观陈述**："该期报道记录了 X" / "档案译载 X 声明全文" / "OCR 文本中可见 X"
3. **聚类不评价**：同一组档案的共性用"上述 N 期共同载入了……" / "该组报道分别记录了……"
4. **不使用预设结论词**：「关键节点」「转折点」「标志着」「开启了」「体现了」「揭示了……的本质」等史学评价词全部回避
5. **不补充档案外的历史背景**：除非档案本身提及该背景，否则不在综述中引入

### 引言：54 期港媒档案的基本面（约 300 字）

陈述档案的客观属性，不做历史评价：
- 54 期分布：China Mail 41 期 + Hong Kong Telegraph 13 期
- 时间跨度：1946-01-11 至 1950-09-12，共 4 年 8 个月
- 入库方式：从每期 100-200KB 整版 OCR 中抽取含「Democratic League / Kuomintang / Communist / 民盟领导人姓名 / 政治协商 / 联合政府」等关键词的段落
- 单期信息密度：抽取段落平均 1500-3000 字英文
- 报道形态：综合性政治新闻为主，含直接采访、官方声明译载、社论分析
