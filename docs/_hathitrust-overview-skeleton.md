## 导论：港媒视角的史料价值

**核心论点**：港媒（China Mail / Hong Kong Telegraph）作为 1940s 远东最重要的英文商业报刊之一，其对民盟的报道构成 **FRUS 美方电报 / DRNH 国民政府机要 / CIA 情报视角之外的「第三方公开舆论场」**——既非国共两党党媒，也非西方政府电报，而是**当时香港商埠市民视角**的实时记录。54 期港媒报道串起 1946–1950 民盟史关键脉络。

**史料独特价值**（三点）：
1. **时间密度**：1946-01 至 1950-09 跨度近 5 年，平均每月有报道
2. **现场性**：港埠记者能近距离接触从内地撤港的民盟领导人（张澜、罗隆基、沈钧儒、章伯钧等），多次报道含直接采访
3. **观察立场**：港埠商业报刊不依附国共任一方，对民盟"被非法化"等事件能呈现两造对照

## 五、1947 民盟"非法化"全过程（5 期，关键节点）

## 综述结语（待写）

港媒（China Mail / Hong Kong Telegraph）54 期 1946–1950 民盟相关报道，构成中国大陆境外档案中**第三方公开舆论场**的完整时间序列。其史料独特价值在于：

1. **填补 FRUS 与 DRNH 之间的"市民观察空白"**
2. **现场细节丰富**（含直接采访、舆论反响、街头集会等）
3. **跨越民盟史最关键的五年**（1946 政协 → 1947 非法化 → 1948 香港重建 → 1949 新政协筹建 → 1950 参政党转型）

研究使用建议：与 FRUS / DRNH / CIA 三源同期档案做"四源对照"研究，可重建民盟史关键节点的多视角真相。
