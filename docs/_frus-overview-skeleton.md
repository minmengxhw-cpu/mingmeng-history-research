## 结语（约 300 字）

陈述档案集合的客观属性：FRUS 299 篇 / 416 页段为 1943-1950 年美方公开外交档案中涉及民盟的部分。其中信息密度最高的为 1946v10（75 篇，国大筹备）、1946v09（61 篇，政协与马歇尔调停）、1947v07（48 篇，联合政府与民盟"非法"）三个分卷，合计 184 篇，占 FRUS 民盟档案的 62%。该集合的史料价值在于：作为公开学术档案，所有引用可经 `history.state.gov` 永久 URL 溯源；matched_terms 命中分布显示罗隆基为出现频率最高的民盟人物（57 次），其次为张澜、章伯钧、沈钧儒、黄炎培。

FRUS 与 DRNH（国民政府内部）、CIA（美方情报）、HathiTrust（港媒）、Hoover（民盟领导人致美方私函）、Wilson（苏方）共同构成民盟史**六源对照档案体系**；FRUS 在六源中代表**美方公开外交视角**，是其他五源最频繁引用对照的基准源。
