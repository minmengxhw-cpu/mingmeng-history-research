# 1947 年 10—11 月民盟「非法化」事件·四源对位证据卡片集

> 民盟历史文献研究库 · 跨源证据卡片库 · 第 001 号事件
> 撰文日期：2026-06-01 · 撰文：研究平台项目组

## 0. 本卡片集的方法学依据

按社科院专家研究方法第 2 条「孤证不举，互证优先」+ 第 5 条「证据卡片」格式落实。
**1947 年 10 月 28 日国民政府宣布民盟"非法"**是民盟史可达成的"多源对位档案密度最高"
时点。本卡片集以 **1947-10-01 至 1947-11-19 共 50 天**为时间窗口，把 **FRUS（美方外交）/
DRNH（国府内部）/ HathiTrust（港媒）/ Hoover（民盟创始人私函）** 四源同期档案逐时点
对位，形成可核验、可追溯的结构化证据卡片。

**卡片字段**（统一）：

| 字段 | 含义 |
|---|---|
| 日期 | 档案的本件日期（公历）|
| 来源 / 档案号 | 平台 + 档案唯一标识 |
| 产出主体 | 谁写的（哪个机构 / 哪位人物） |
| 原文摘录 | 档案中关键英文/繁体中文片段（已核对） |
| 关键词 | 用于跨源检索的标签 |
| 历史背景 | 事件链中的位置 |
| 人物关系 | 涉及的民盟与各方人物及其角色 |
| 跨源对应 | 同时点其他源档案对位 |
| 可信度 | 1—5 级（1 最低、5 已多源互证）|
| 研究价值 | 这张卡片能回答什么问题 |
| 后续检索 | 还需查什么档案补强 |
| 证据等级 | L1 一手原档已核 / L2 已下待核 / L3 目录元数据 / L4 LLM 精读 |

下面按时间线呈现 17 张关键时点卡片。

## 四、卡片集总览与方法论小结

### 4.1 50 天事件的四源对位密度统计

| 源 | 时段内档案数 | 主要时点 |
|---|---:|---|
| FRUS | ~14 篇 | 1947-10-01 → 11-19 |
| DRNH | ~13 篇 | 1947-10-04 → 12-24 |
| HathiTrust | 4 期 | 10-29 / 10-30 / 11-05 / 12-23 |
| Hoover | 1 件 | 11-01 |
| **合计** | **~32 篇** | **50 天内**（峰值在 10-28 至 11-07 共 10 天）|

### 4.2 本卡片集的核心发现

1. **三源同日对位的最高密度日 = 1947-11-05**（FRUS + DRNH + HathiTrust 同日，加 Hoover 4 天前的档案在记忆窗口内）— 是民盟史可达成的多源最完整覆盖
2. **唯一的"民盟方主动陈述"档案 = T13 张君劢 1947-11-01 致 Marshall 函**（Hoover），证据等级 L1
3. **国府决策链最早的内部档案 = T05 俞济时 1947-10-22 呈"解散民盟办理情况"**（早于 10-28 公开宣布 6 天）
4. **三处时间冲突 / 单源孤证需进一步互证**：T02 杜斌丞被枪决日期、T06 蒋作战会报"有所指示"内容、T15 王世杰"政府不再追究"无 DRNH 对位
5. **海外取缔意图 = T17 国府要求港英取缔民盟**（蒋亲自指示），是民盟史既有研究中罕见提及但极具意义的档案

### 4.3 按方法论的研究产出

本卡片集可直接支持以下研究：

- **决策性质问题**（T05 + T06 + T12）：民盟非法化是蒋独断 / CC 系主推 / 多方共识？
- **美方介入度问题**（T03 + T04 + T11 + T12 + T13）：美国务院最高层介入对国府最终采取"温和取缔"的因果作用？
- **民盟方应对策略问题**（T04 + T07 + T11 + T13 + T15）：民盟在非法化前后的对美求援、对国府谈判、自行解散决定的内部决策链？
- **海外取缔扩展问题**（T17）：国府将取缔扩展至香港的尝试及英方反应？
