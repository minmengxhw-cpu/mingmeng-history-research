# 民盟历史文献研究库

民盟历史文献研究库是一个围绕 1941-1950 年中国民主同盟史建设的个人学术研究平台。平台把 FRUS、CIA FOIA、Wilson Center、Hoover、HathiTrust / Internet Archive、DRNH 等来源中的相关原始档案统一整理为“原文全文 + 中文译文 + 页码引用 + 来源链接”的证据链，方便进行人物、事件、机构和跨国档案视角的比对研究。

## 平台定位

本项目不是二手资料汇编，也不是结论先行的叙事网站。它的核心任务是把散落在不同档案机构、数字典藏和公开出版物中的民盟相关材料结构化，形成可以检索、可以复核、可以引用的研究底座。

## 当前内容

- 六个档案平台已进入统一文档库，并按“核心文献、相关文献、人物关联、背景材料、前台不展示”分级。
- 每篇文档保留原始来源链接、英文或原始文本、中文译文、页面片段和学术引用格式。
- 研究论文页面把平台级史料价值、关键样本和已引用原档连接起来。
- 资料包页面提供各平台 ZIP 包下载，便于离线校读和阶段性备份。

## 使用方式

研究时可先从“研究论文”了解各平台材料的史学价值，再进入“全部文档”或“搜索”按人物、年份、平台、文献分级筛选。正式写作时，应从文档页进入引用卡片，并回到原始档案链接核对页码、日期、标题和原文。

## 学术声明

平台中的中文译文用于研究阅读和线索定位，正式发表仍应以原始档案文本、原始影像或档案机构页面为准。机器翻译与人工校订状态会在页面中保留标记；对主题不符、误命中或名称相似但不属于中国民主同盟研究范围的材料，统一进入“前台不展示”清单，保留可追溯剔除理由。
