# 1946 年 5 月广西取缔民盟事件·跨源证据卡片集

> 民盟历史文献研究库 · 跨源证据卡片库 · 第 002 号事件
> 撰文日期：2026-06-01 · 撰文：研究平台项目组

## 0. 本卡片集的方法学依据 + 事件定位

按方法论第 2 条「孤证不举，互证优先」+ 第 5 条「证据卡片」格式落实。

**本事件的核心方法学价值**：1946-05 广西取缔民盟事件是 DRNH 国府打压方档案中
**蒋介石本人在最短时间窗口内连续下达取缔民盟电令最密集的一组档案**——3 天内
5 件亲笔电令，且发生在民盟全国"非法化"（1947-10-28）前 **17 个月**。本事件是
研究"国府对民盟政策从1946-01-19 蒋'沆瀣一气'定性 → 1946-05 地方'取缔实验'
→ 1947-10 全国'非法化'"链条的关键节点。

**跨源对位特点**：DRNH 国府内部档案密集（8 件），其他源同期对位稀缺
（FRUS / Hoover / Wilson 无对位档案；HathiTrust 仅 1947-08-04 跨年 14 个月
的事后港媒间接报道）。这是**典型的"DRNH 国府内部档案孤证"事件**——
打压方行动在公开舆论场和美方外交场视野外完成。

**卡片字段**（沿用第 001 号事件标准）：来源 / 产出主体 / 原文摘录 / 关键词 /
历史背景 / 人物关系 / 跨源对应 / 可信度 / 研究价值 / 后续检索 / 证据等级。

## 六、卡片集总览与方法论小结

### 6.1 本事件的档案分布

| 源 | 时段内档案数 | 主要时点 |
|---|---:|---|
| DRNH | 9 篇 | 1946-01-19 / 05-01 / 05-20 / 05-22 ×4 / 05-31 / 1946-05-22 民盟反向呈件 |
| FRUS | 0 | （马歇尔调停档案侧重中央层级，未捕捉广西地方）|
| HathiTrust | 1 | 1947-08-04 跨年事后港媒间接报道 |
| Hoover / Wilson / CIA | 0 | （时段过早，三源均无）|
| **合计** | **10 篇** | DRNH 独大，**典型的"国府打压方内部孤证事件"** |

### 6.2 本卡片集的核心发现

1. **5 天内 5 件蒋本人亲电**（T03—T07）：1946-05-20、5-22×4——蒋本人对一省民盟地方分支的处置力度异常密集
2. **同日反差档案**（T08）：蒋下达取缔民盟电令的同一天（5-22），民盟领导人向蒋呈递东北调停方案——"中央调解 vs 地方镇压"双轨态势
3. **曾毅案的双轨调查**（T06 + T07）：蒋对三青团干事长曾毅加入民盟一案，同日通过省政府（T06）和党组织（T07）两条线同时下令调查——反映**蒋对"民盟渗透国民党内"的恐惧**
